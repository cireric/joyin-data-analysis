## 术语说明

| 术语 | 说明 |
|------|------|
| **周期** | `5h` = 5小时 | `w` = 周 | `mo` = 月 |
| **额度倍率** | 当前周期额度 ÷ 包月价格（倍率越高，性价比越高） |
| **TPS** | Tokens Per Second，每秒生成Token数 |
| **RPM** | Requests Per Minute，每分钟请求数 |
| **上下文长度** | 模型支持的最大输入Token数 |
| **Coding Plan** | 面向编程场景的订阅套餐，通常有请求次数限制 |
| **Token Plan** | 基于Token消耗量的订阅套餐 |
