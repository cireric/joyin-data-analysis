# 多周期数据分析报表设计

## 概述

扩展 `scripts/run_analysis.py` 支持多种周期类型：年度、季度、月度、自定义时间范围。

## 需求

### 周期类型

| 类型 | 输入格式 | 文件名示例 |
|------|---------|-----------|
| 年度 | `2026` | `销售汇总表-2026.xlsx` |
| 季度 | `2026Q1` | `销售汇总表-2026Q1.xlsx` |
| 月度 | `2026.03` | `销售汇总表-2026.03.xlsx` |
| 自定义 | `2026.04.13~2026.04.19` | `销售汇总表-2026.04.13~2026.04.19.xlsx` |

### 识别规则

按优先级顺序匹配：

1. 包含 `~` → 自定义范围
2. 匹配 `^\d{4}Q[1-4]$` → 季度
3. 匹配 `^\d{4}\.\d{2}$` → 月度
4. 匹配 `^\d{4}$` → 年度
5. 无法识别 → 报错并提示支持的格式

### 文件存在性检查

执行前检查 `data/` 目录中的数据文件：

- 文件不存在时显示明确错误信息
- 错误格式：`错误: 未找到数据文件: data/销售汇总表-2026.xlsx`
- 检查通过显示：`数据源: 销售汇总表-2026.xlsx, 销售汇总表-2025.xlsx`

## 命令行接口

### 基本用法（自动识别）

```bash
make analyze c=2026 p=2025                    # 年度
make analyze c=2026Q1 p=2025Q1                # 季度
make analyze c=2026.03 p=2026.02              # 月度
make analyze c=2026.04.13~2026.04.19 p=2026.04.06~2026.04.12  # 自定义
```

### 强制指定类型（可选）

```bash
make analyze c=2026.03 p=2026.02 type=quarter
```

### 其他参数

```bash
make analyze c=2026.03 p=2026.02 group=督导人员
make analyze c=2026.03 p=2026.02 columns=总杯数 总金额
make analyze c=2026.03 p=2026.02 title=自定义标题
```

## 报表标题生成

| 类型 | 标题格式 |
|------|---------|
| 年度 | `{prev}-{curr}年度销售分析` |
| 季度 | `{prev}-{curr}季度销售分析` |
| 月度 | `{prev}-{curr}月度销售分析` |
| 自定义 | `{prev_short}与{curr_short}销售分析` |

自定义范围标题简化显示：
- 输入：`2026.04.13~2026.04.19`
- 标题中显示：`04.13~04.19`

## 实现要点

### 新增模块

1. **周期识别器** (`PeriodParser`)
   - 解析输入字符串
   - 返回周期类型和标准化格式
   - 生成文件名

2. **文件验证器** (`FileValidator`)
   - 检查数据文件是否存在
   - 返回明确的错误信息

### 修改文件

- `scripts/run_analysis.py` - 主脚本
- `Makefile` - 更新命令示例

### 向后兼容

保持原有参数格式兼容：
- `-c/--current` 和 `-p/--previous` 参数名不变
- 默认值和可选参数保持不变
- 现有月度格式 `2026.03` 继续正常工作

## 错误处理

| 场景 | 错误信息 |
|------|---------|
| 无法识别格式 | `错误: 无法识别周期格式 'xxx'。支持的格式: 2026, 2026Q1, 2026.03, 2026.04.13~2026.04.19` |
| 当前期文件不存在 | `错误: 未找到当前期数据文件: data/销售汇总表-2026.xlsx` |
| 对比期文件不存在 | `错误: 未找到对比期数据文件: data/销售汇总表-2025.xlsx` |
| 两文件都不存在 | 分别列出两个缺失文件 |

## 测试用例

1. 年度：`c=2026 p=2025` → 文件 `销售汇总表-2026.xlsx`, `销售汇总表-2025.xlsx`
2. 季度：`c=2026Q1 p=2025Q1` → 文件 `销售汇总表-2026Q1.xlsx`, `销售汇总表-2025Q1.xlsx`
3. 月度：`c=2026.03 p=2026.02` → 文件 `销售汇总表-2026.03.xlsx`, `销售汇总表-2026.02.xlsx`
4. 自定义：`c=2026.04.13~2026.04.19 p=2026.04.06~2026.04.12` → 对应文件
5. 无效格式：`c=abc p=def` → 显示支持的格式列表
6. 文件不存在：显示明确错误信息
